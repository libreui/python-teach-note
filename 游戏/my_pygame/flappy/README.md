# Flappy Bird Game

A Python implementation of the classic Flappy Bird game using Pygame.

## Features (功能列表)

### ✅ Completed (已完成)

- [x] 游戏窗口初始化
- [x] 背景图片滚动
- [x] 地面无限滚动
- [x] 小鸟跳跃控制
- [x] 小鸟翅膀煽动动画
- [x] 小鸟旋转效果（跳跃上扬/下落低头）
- [x] 落地碰撞检测
- [x] 游戏结束状态设置
- [x] 管道生成与随机高度
- [x] 管道无限滚动
- [x] 管道碰撞检测
