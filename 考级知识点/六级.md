 # Python六级考试知识点掌握
## 1 面向对象编程
- 类的定义：class 类名()，命名规范（简洁有意义、大驼峰）
- 初始化方法：__init__(self, 参数1, 参数2...)
- 实例创建：对象名 = 类名(参数)
- 实例属性：self.属性名 = 值
- 实例方法：def 方法名(self):
- 类的使用：必须先定义后引用，创建实例才能调用方法
- 方法调用：实例.方法名()
- 类的导入：from 模块 import 类名 / import 模块
- 继承与super()：调用父类方法
- 方法重写：子类重写父类同名方法

## 2 JSON模块操作
- json.loads()：JSON字符串 → Python对象（字典/列表）
- json.dumps()：Python对象 → JSON字符串
- json.load()：从JSON文件读取 → Python对象
- json.dump()：Python对象 → 写入JSON文件
- 文件打开模式：读取用文本模式，非二进制

## 3 CSV文件操作
- csv.reader()：按行读取，返回列表
- csv.DictReader()：以字典形式读取，按字段名访问
- csv.writerow()：写入一维列表
- 一维数据存储：列表转二维嵌套列表写入
- 分隔符默认逗号，无需强制指定delimiter

## 4 Tkinter图形界面
- 窗口创建：tk.Tk()、title()、geometry()
- 主循环：mainloop()，可不设置geometry自动显示
- 常用控件
  - Label：文本标签，config()修改属性
  - Entry：输入框，show设置密码显示字符
  - Button：按钮，command绑定点击事件
  - Scale：滑块，orient方向、length长度、command回调
  - OptionMenu：下拉列表
  - Listbox：列表框
  - Scrollbar：滚动条
  - Checkbutton：复选框
- 布局：pack()，side设置对齐方向（left/top等）
- 事件绑定：command=函数名（不加括号）

## 5 文件操作
- 打开模式
  - r：只读
  - w：写入，清空原有内容
  - a：追加，在末尾写入
  - r+：读写
  - rb/wb/ab：二进制读写
- 读取方法
  - read()：读取全部内容
  - readline()：读取一行
  - readlines()：读取所有行，返回列表
- 写入方法：write()
- 文件指针：seek(n) 移动到第n字节位置
- with语句：自动关闭文件，异常安全
- 文件路径：支持相对路径与绝对路径
- 编码问题：读写编码不一致导致乱码
- 行反转：readlines()读取后reverse()/reversed()倒序写入

## 6 SQLite数据库操作
- 执行顺序：import → connect → cursor → 操作 → close cursor → close conn
- 连接：sqlite3.connect(文件名)，不存在则自动创建
- 游标：conn.cursor()
- 创建表：CREATE TABLE IF NOT EXISTS，PRIMARY KEY主键（唯一非空）
- 数据类型：INTEGER、TEXT、REAL、BLOB（二进制）
- 插入
  - execute()：单条插入
  - executemany()：批量插入
- 查询
  - fetchone()：取一条
  - fetchmany(n)：取n条
  - fetchall()：取全部
- 更新：UPDATE 表 SET 字段=值 WHERE 条件
- 删除：DELETE FROM 表 WHERE 条件
- 事务：commit()提交，增删改必须提交
- 关闭：先关游标，再关连接

## 7 NumPy数值计算
- 数组创建
  - np.array()
  - np.arange(start, stop, step)
  - np.zeros(n)：创建全0数组
- 数组属性：索引（支持负索引）、形状
- 形状修改
  - reshape()：不修改原数组，元素总数不变
  - resize()：修改原数组
- 元素运算：数组间逐元素加减乘除
- 统计函数
  - sum()：求和
  - mean()：平均值
  - max()：最大值
  - size()：元素个数
- 特点：比普通列表内存小、运算快，支持多维矩阵

## 8 Matplotlib数据可视化
- figure()：创建画布
- plot()：绘制折线图
- scatter()：散点图
- hist()：直方图
- title()：设置图表标题
- legend()：添加图例，loc设置位置（upper right等）
- 线条样式：颜色（r/g）+ 线型（-/--）组合
- 中文设置：plt.rcParams["font.sans-serif"] = "SimHei"
- show()：显示图形
